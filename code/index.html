HTTP

- Hypertext Transfer Protocol -

What the h[ttp] is this?

To summarize what HTTP is in just once sentence, it would go something like this: It's the foundation of data communication for the World Wide Web (www). HTTP is an abbreviation of Hypertext Transfer Protocol and the development of HTTP was originally initiated by Tim Berners-Lee back in 1989. This was the guy that invented the World Wide Web (side note: quite an awesome acchievement!) So, basically, HTTP is about the communication between web servers (most often computers in the cloud) and clients (for example browsers, such as Safari and Chrome). Everytime you are for example clicking a link on a webpage or submitting a form, HTTP is being used. This is done by a so called request and response cycle.
